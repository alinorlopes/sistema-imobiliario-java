# 💰 Sistema de Financiamento Imobiliário — Java

Este projeto é um sistema simples em Java que simula o cálculo de financiamento imobiliário. O usuário informa os valores principais (como valor do imóvel, entrada, taxa de juros e prazo), e o sistema retorna os dados do financiamento.

## 🎯 Objetivo

- Praticar encapsulamento, abstração, herança e polimorfismo
- Estruturar um projeto Java limpo e modular
- Simular um caso real de aplicação de lógica financeira

## ⚙️ Funcionalidades

- Entrada de dados do imóvel e condições do financiamento
- Cálculo da parcela mensal com base em juros compostos
- Exibição do valor total financiado e custo final

## 🛠️ Tecnologias

- Java (recomendado: versão 8 ou superior)
- IDE sugerida: IntelliJ IDEA, Eclipse ou VS Code com extensão Java

## ▶️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/alinorlopes/sistema-imobiliario-java.git
2. Abra o projeto em sua IDE Java
3. Execute o arquivo Main.java
4. Siga as instruções no terminal para inserir os valores
   
📌 Exemplo de Resultado
Valor do imóvel: 300000  
Entrada: 60000  
Taxa de juros anual: 8  
Prazo (em anos): 20  


Resultado:
Parcela mensal: R$ 2.005,34
